# Swarm Command

Activate swarm mode with all 11 specialized agents running in parallel.

**Usage:** `/swarm` or press `Cmd+E`

**Configuration:** See `.cursor/config/swarm-config.json`

**Agents Activated:**
- Security Agent
- Performance Agent
- Testing Agent
- Documentation Agent
- Migration Agent
- Dependencies Agent
- Bug Hunter Agent
- Accessibility Agent
- API Agent
- Database Agent
- Beta Tester Agent

**Prefix:** All outputs prefixed with "CTO Eleven MCP Swarm"
